# 🌿 InAmigos Foundation — NGO Awareness Webpage

> **"Creating Change, One Community at a Time"**  
> An awareness webpage for **InAmigos Foundation**, a registered Section 8 NGO working across food security, education, environment conservation, animal welfare, and youth empowerment in India.

## 🔗 Live Preview

> Open `index.html` directly in any modern browser to view the webpage locally.

## 📋 About the Project

This project was created as part of an **internship task** to spread awareness about the **InAmigos Foundation** and its ongoing social initiatives. The webpage is designed to be clean, modern, interactive, and informative — giving visitors a full picture of the NGO's mission, projects, and how they can get involved.

## 🌟 Features

| Feature | Description |
|---|---|
| 🌙 **Dark / Light Mode** | Toggle between themes with preference saved in localStorage |
| 📱 **Fully Responsive** | Works on mobile, tablet, and desktop |
| 🔍 **Project Filter** | Filter 6 projects by category (Humanitarian, Education, Environment, etc.) |
| 📖 **Accordion Details** | Expand each project card for detailed social impact info |
| 📊 **Animated Counters** | Stats count up from 0 when scrolled into view |
| 🎞️ **Testimonial Slider** | Auto-playing slider with touch/swipe support |
| 📝 **Volunteer Form** | Real-time validation + success state with unique reference code |
| ✨ **Scroll Animations** | Elements reveal smoothly as you scroll |
| 🍔 **Hamburger Menu** | Mobile-friendly collapsible navigation |

## 🛠️ Tech Stack

- **HTML5** — Semantic structure, accessibility (ARIA labels)
- **CSS3** — CSS Variables, Grid, Flexbox, Keyframe animations, Glassmorphism
- **Vanilla JavaScript (ES6+)** — IntersectionObserver, event delegation, localStorage
- **Google Fonts** — Inter + Outfit typefaces

> ⚡ No frameworks, no dependencies, no build tools needed — just open and run!

---

## 💻 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/viveksharma151/NGO-awareness-webpage.git
   ```

2. **Navigate to the folder:**
   ```bash
   cd NGO-awareness-webpage
   ```

3. **Open in browser:**
   - Simply double-click `index.html`  
   - OR right-click → *Open with* → Chrome / Edge / Firefox

> No server or npm install required!
